# Camping 2026-Jul

Standalone deployment of the camping trip checklist — passcode-gated CRUD
list where the group claims what they're bringing to Jubilee Park
Campground, Wizard Lake, Edmonton AB (July 24-26, 2026).

Backed by the same Supabase project as the original portfolio-embedded
version. See `supabase/` in the [ojdh.github.io](https://github.com/ojdh/ojdh.github.io)
repo for the database schema this depends on.
